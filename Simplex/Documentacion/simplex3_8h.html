int  f,
int  c,
int  noColumna 
)

Busca cuál es la fila que tenga el dato de menor valor, esto para sacar el pivote.

Dentro de la funcion se hara una división de columnas se dividirá la columna actual entre los valores de la columna pivote. Los valores que resulten de la división serán guardados en una matriz auxiliar, posteriormente se buscara cual es dato de menor valor.

Parámetros
mVariable para la matriz.
fVariable para el número de filas.
cVariable para el número de columnas.
noColumnaVariable para almacenar el número de la columna con el valor menor.
Devuelve
Regresa "noFila" que es una variable de tipo entero la cual contiene el número de la fila con el dato menor.
